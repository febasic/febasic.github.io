Единицы измерения

vh 1vh соответствует 1% высоты viewport
vw 1vw соответствует 1% ширины viewport
vmin 1vmin соответствует меньшему из 1vh, 1vw
vmax 1vmax соответствует большему из 1vh, 1vw
rem 1rem соответствует значению свойства font-size корневого элемента

meta viewport

Responsive Web Design

Резиновая верстка

В резиновой верстке размеры для составных элементов layout'а указываются в процентах, то есть они меняют свои размеры в зависимости от размеров окна браузера.

Адаптивная верстка

Адаптивная верстка - это подход, основной идеей которого является корректное отображение интерфейса на конкретных, фиксированных разрешениях (например: 576px, 768px, 992px, 1200px). Переход между брэйкпоинтами осуществляется рывками, а не плавно.
Разница между фиксированной, резиновой, адаптивной и отзывчивой вёрсткой

Mobile First

Mobile First - это подход разработки пользовательского интерфейса в котором изначально осуществляется разработка для мобильных устройств, а затем для широкоформатных дисплеев.

Graceful Degradation

Graceful Degradation (постепенная деградация) - это подход разработки, ориентированный на работу в самых современных браузерах. При этом в более старых браузерах также важно сохранить функциональсть приложения.

Progressive Enhancement

Progressive Enhancement (прогрессивное улучшение) - это подход разработки идея которого заключается в создании простого и функционально интерфейса на первом этапе разработки и постепенном улучшении на следующих итерацияx.

Media Queries

Медиа-запросы для разных типов
устройств

all | screen | print | speech 
      @media screen, print {
        body {
          background-color: black;
        }
      }

Медиа-запросы для разной
ориентации устройств

landscape | portrait 
      @media (orientation: landscape) {
        body {
          background-color: black;
        }
      }

Медиа-запросы для разных размеров
устройств

min-width | max-width | min-height | max-height 
      @media (min-width: 640px) {
        body {
          background-color: black;
        }
      }

Медиа-запросы для устройств
c высокой плотностью пикселей

      @media (-webkit-min-device-pixel-ratio: 2),
             (min-resolution: 192dpi) {
        body {
          background-image: url('image_2x.png');
        }
      }

Бонус: @supports

@supports позволяет проверить поддержку отдельных css свойств либо технологий в браузере. 
      @supports (background-clip: text) {
        .section {
          background-color: black;
        }
      }
