Комбинаторы

Комбинатор Описание Пример
, выбор нескольких элементов div, span { }
' ' выбор всех вложенных элементов nav a { }
> выбор дочерних элементов первого уровня вложенности ul > li { }
~ выбор последующих соседей p ~ span { }
+ выбор элемента, который находится непосредственно после указанного p + span { }

Селекторы

Селектор Описание
[href] выбор всех элементов у которых есть заданный атрибут
[title='link'] выбор всех элементов у которых есть заданный атрибут c заданным значением
[title*='nk'] выбор элементов по частичному совпадению значения атрибута
[title^='li'] выбор элементов у которых первые символы значения атрибута совпадают с указанным значением
[href$='txt'] выбор элементов у которых последние символы значения атрибута совпадают с указанным значением

Псевдоклассы

Псевдокласс Описание
:first-child выбор первого потомка
:last-child выбор последнего потомка
:hover устанавливает стили при наведении курсора мыши
:focus устанавливает стили в состоянии фокуса
:active устанавливает стили элемента в активном состоянии (нажатая кнопка мыши)

Сколько весят селекторы?

!important

!important повышает приоритет стилей 
      a {
        color: red !important;
      }

Задачка

      <a style='color:red' href="//google.com">Какого я цвета?<a>
      a[href] {
        color: blue;
      }
      

        a[href] {
          color: green !important;
        }
      

      

        a {
          color: black !important;
        }
