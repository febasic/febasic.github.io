HTML (Hypertext Markup Language)

HTML — это язык разметки, разработанный для создания веб-страниц.
С помощью HTML мы можем добавить на страницу:
  • контент (текст, изображения, видео, аудио);
  • элементы пользовательского интерфейса (кнопки, ссылки, формы и т.д.);
  • meta-информацию.
Современная спецификация HTML.

Немного истории

HTML был изобретён Тимом Бернерсом-Ли, физиком из исследовательского института CERN.

Первая версия языка HTML была опубликована в 1991 году
и состояла из 18 тегов.

Структура HTML документа

      
      <!DOCTYPE html>
      <html lang='en'>
        <head>
          <!-- этот раздел предназначен для 'служебной' информации -->
        </head>
        <body>
          <!-- этот раздел предназначен для содержимого страницы (контента) -->
        </body>
      </html>

Тег

Тег - это элемент HTML разметки.
Имя тега оборачивается специальными символами '<''>'
Теги бывают одинарные и двойные.
Примеры одинарных тегов: <img>, <br>
Двойной тег состоит из открывающего тега и закрывающего тега:
Пример двойного тега: <h1>Главный заголовок</h1>

Атрибуты

Атрибут — это специальное слово внутри открывающего(!) тега необходимое для определения характеристик HTML элемента.
Атрибут состоит из имени и значения.
Пример: <a href='https://google.com'>Google</a>
Имя: href
Значение: https://google.com

meta теги

meta теги - это теги необходимые для передачи служебной информации (такой которая не рендерится на странице).
Примеры:
<title>Заголовок страницы вкладки браузера</title> <meta name='description' content='Описание страницы'>

Возможности мета тегов

С помощью meta тегов можно:
  • установить заголовок, описание, ключевые слова (SEO информация);
  • добавить иконку (favicon) сайта;
  • настроить вид превьюшки для шеринга через соц.сети ( Open Graph );
  • настроить viewport (необходимо для корректного отображения страницы на мобильных устройствах).

Разметка текста с помощью HTML

Обозначение Назначение Примечание
<h1> главный заголовок страницы обязательно (!) только один h1 на странице
<h2>...<h6> другие заголовки по убыванию степени важности. <h2>-более важные <h6>-наименее важные
<p> абзац (параграф) Определяет текстовый абзац.
<em> акцентированный текст отображается курсивным начертанием
<i> курсивный текст (italic) отображается курсивным начертанием
<strong> важный текст отображается жирным начертанием
<b> жирный текст (b) отображается жирным начертанием
<mark> выделенный текст отображается как-то так
Важно: не используйте теги для стилизации! Для этого есть CSS.

Блочный элемент vs строчный элемент

Блочный элемент занимает 100% ширины своего родителя.
Пример:
<div></div>
Строчный элемент изменяет ширину в зависомотсти от количества контента
<span>Мало контента</span>
<span>Немножко больше контента</span>
