SVG

Система координат

Начало координат - верхний левый угол документа.

Viewbox

Атрибут viewBox определяет расположение и размеры окна отображения SVG. 
      <svg viewBox='0 0 100 100'>
        <rect x='0' y='0' width='100%' height='100%'/>
      </svg>

Заливка и обводка

      fill: #00ff00;
      stroke: #0000ff;
      stroke-width: 4;

      fill-opacity: 0.5;
      stroke-opacity: 0.5;

Строительные блоки

Строительными блоками в SVG графике являются примитивные фигуры, такие как: линия, прямоугольник, окружность, полигон, путь и др.

Line

      <svg viewBox='0 0 100 100'>
        <line x1='0' y1='0' x2='100' y2='100' stroke='black' />
      </svg>

Rectangle

      <svg viewBox='0 0 200 100'>
        <rect x='0' y='0' width='50%' height='100%'/>
        <rect x='50%' y='0' rx='30' ry='30' width='50%' height='100%'/>
      </svg>

Circle

      <svg viewBox='0 0 100 100'>
        <circle cx='50' cy='50' r='50'/>
      </svg>

Ellipse

      <svg viewBox='0 0 200 100'>
        <ellipse cx='100' cy='50' rx='100' ry='50'/>
      </svg>

Polyline

      <svg viewBox='0 0 200 100'>
        <polyline points='0, 0 40,40, 60,90, 70,30, 120,70, 200,0' 
        stroke='#244c8e' fill='none'/>
      </svg>

Polygon

      <svg viewBox='0 0 200 100'>
        <polygon points='0, 90 40,40, 60,90, 70,30, 120,70, 200,100, 0,90'
        stroke='#244c8e' fill='#244c8e'/>
      </svg>

Path

      <svg viewBox='0 0 200 100'>
        <path d="M10 10 C 10 20, 10 190, 150 20"
        stroke="#244c8e" fill="transparent"/>
      </svg>

Text

      <svg viewBox='0 0 200 100'>
        <text x='10' y='20'>Lorem, ipsum dolor sit
        amet consectetur adipisicing elit.</text>
      </svg>

TextPath

      <svg width='400' height='200' viewBox='0 0 200 100'>
        <path id='my_path' d='M10 10 C 10 20, 10 190, 200 20' fill='transparent'/>
        <text>
          <textPath xlink:href='#my_path'>svg text path</textPath>
        </text>
      </svg>

Как создать clipPath

      <svg>
        <clipPath id='my-clip-path'
                  clipPathUnits='objectBoundingBox'>
          <path d='M0,0 C1,0,1,1,0,1'></path>
        </clipPath>
      </svg>

Как применить clipPath

      .image {
        clip-path: url(#my-clip-path);
      }

Как применить clipPath

            <svg>

Как создать SVG маску

      <svg width='100%' height='100%'>
        <defs>
          <mask id='mask' x='0' y='0' width='100%' height='100%'>
            <rect fill='white' x='0' y='0' width='100%' height='100%'/>
            <text x='50%' y='50%' fill='black' text-anchor='middle'>SVG</text>
           </mask>
        </defs>
        <rect fill='white' x='0' y='0' width='100%' height='100%' />
      </svg>

Как применить SVG маску

      svg rect {
        mask: url(#mask);
      }
