Что такое frontend?

Frontend - визуальная составляющая и функционал на стороне клиента (браузера).

Из чего состоит frontend

HTML CSS JavaScript
Разметка, контент Внешний вид Динамика, функционал

Верстальщик (WEB UI Engineer)

Зона ответственности Миссия верстальщика: разработка удобных, эффективных и доступных интерфейсов.

Современный frontend это:

  • разработка сайтов (demo 1 2 3 4 5);
  • разработка SPA (Single Page Apps React Vue Angular);
  • разработка мобильных приложений ( React Native Native Script Web View);
  • разработка игр;
  • разработка PWA (Progressive Web Apps);
  • разработка desktop приложений (Electron).
