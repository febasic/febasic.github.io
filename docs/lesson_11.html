Transition

CSS свойство transition позволяет реализовать плавный переход между двумя состояниями элемента.

Пример

            .button {
              transition: all 0.3s;
            }
            .button:hover {
              color: white;
              background-color: navy;
            }

transition-property

Свойство transition-property устанавливает какие именно css свойства должны изменяться плавно. Как правило transition можно применить ко всем свойствам, которые имеют числовое значение, но есть исключения. 
      .transition {
        transition-property: color, background-color;
      }

transition-duration

Свойство transition-duration устанавливает в течении какого отрезка времени будет произведен переход из одного состояния в другое. 
      .transition {
        transition-duration: 0.5s;
      }

transition-timing-function

Свойство transition-timing-function устанавливает функцию с помощью которой будут вычислены промежуточное состояния во время перехода. 
      .transition {
        transition-timing-function: linear | ease-in;
        transition-timing-function: cubic-bezier(0,1.69,.93,-1.37);
      }
https://cubic-bezier.com/

transition-delay

Свойство transition-duration устанавливает временной отрезок на который будет отложено начало плавного перехода. 
      .transition {
        transition-delay: 0.5s;
      }

Домашка (опционально)

С помощью свойства transition реализовать плавные переходы между состояниями default и hover | focus | active для кнопок и ссылок.
