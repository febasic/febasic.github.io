Свойство margin

Устанавливает внешний отступ. 
            div {
             margin-top: 10px;
             margin-right: 20px;
             margin-bottom: 30px;
             margin-left: 40px;
            }
          
            div {
             margin: 10px 20px 30px 40px;
            }

Хак

      div {
       width: 100px;
       height: 100px;
       margin-right: auto;
       margin-left: auto;
      }

Свойство padding

Устанавливает внутренний отступ. 
            div {
             padding-top: 10px;
             padding-right: 20px;
             padding-bottom: 30px;
             padding-left: 40px;
            }
          
            div {
             padding: 10px 20px 30px 40px;
            }

Особенности

  • схлопывание margin;
  • выпадение margin из родителя;
  • margin и pagging для inline элементов.

Задачка

      <div>
        <div style='padding-bottom: 20px;'></div>
        <div style='padding-top: 30px;'></div>
      </div>
Чему равна высота родительского блока?

Задачка 2

      <div>
        <div style='margin-bottom: 20px;'></div>
        <div style='margin-top: 30px;'></div>
      </div>
Чему равна высота родительского блока?

Свойство border

Устанавливает рамку элемента. 
            div {
             border-width: 10px;
             border-style: solid;
             border-color: #0f0;
            }
          
            div {
             border: 10px solid #0f0;
            }

Свойство border-radius

Устанавливает радиус скругления уголков рамки.
border-radius: 0;
border-radius: 10px;
border-radius: 50%;
border-radius: 50% 0 50% 0;

Свойства width, height, box-sizing

Свойство width/height устанавливает ширину/высоту элемента.
Свойство box-sizing применяется для изменения алгоритма расчёта ширины и высоты элемента.
box-sizing: content-box | border-box

Display inline | block | inline-block

inline Элемент отображается как строчный.
block Элемент показывается как блочный.
inline-block Это значение генерирует блочный элемент, который обтекается другими элементами веб-страницы подобно строчному элементу.
