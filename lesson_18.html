Оптимизация графики

План

  1. Инструменты для оптимизации графики.
  2. Оптимизация загрузки изображений.

Зачем оптимизировать графику?

  1. для оптимизации скорости загрузки;
  2. для формирования лояльной аудитории;
  3. для более высокого ранжирования в поисковой выдаче.

Как оптимизировать графику?

  1. использовать графические форматы по назначению;
  2. использовать прогрессивные форматы графики (webp, jpeg 2000, avif)*;
  3. использовать инструменты для сжатия графических файлов;
  4. использовать в проекте адаптивную графику (picture, media queries, image-set.)
Стоит предусмотреть fallback в формате jpeg/png для браузеров, которые не умеют работать с webp

Популярные инструменты для сжатия графических файлов

  • squoosh
  • tinypng
  • svgomg
