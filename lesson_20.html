Структура

Свойства для grid контейнера

Display

      .grid {
        display: grid | inline-grid;
      }

fr

grid-gap gap

Примеры:
gap: 10px; row-gap: 10px; column-gap: 10px;

grid-template-columns

Свойство grid-template-columns задает количество колонок в сетке а также их ширину.
Примеры:
grid-template-columns: 1fr 1fr 1fr;
grid-template-columns: repeat(3, 1fr)
grid-template-columns: 80px 1fr 500px;
grid-template-columns: 1fr 80% 1fr;

grid-template-rows

Свойство grid-template-rows задает количество строк в сетке а также их высоту.
Примеры:
grid-template-rows: 1fr 1fr 1fr;
grid-template-rows: repeat(3, 1fr)
grid-template-rows: 80px 1fr 500px;
grid-template-rows: 1fr 80% 1fr;

grid-template-areas

Свойство grid-template-areas задает названия grid областей.
Пример:
      grid-template-areas:
        "header header header"
        "nav main aside"
        "footer footer footer";

grid-template

Свойство grid-template объединяет свойства: grid-template-columns grid-template-rows grid-template-areas
Пример: 
      grid-template:
        "header header header" 80px
        "nav main aside" 1fr
        "footer footer footer" 160px / 1fr 4fr 1fr;

minmax()

Пример:
minmax(400px, 50%)
minmax(min-content, 200px)
minmax(max-content, 200px)

repeat()

Пример:
repeat(12, 1fr)
repeat(12, 300px 1fr)

Свойства для grid элемента

grid-column

Свойства grid-column-start, grid-column-end определяют положение элемента в сетке по горизонтали, посредством указания номеров линий. Свойство grid-column объединяет grid-column-start, grid-column-end.
Пример:
grid-column-start: 4;
grid-column-end: 5;
grid-column: 4/5;

grid-row

Свойства grid-row-start, grid-row-end определяют положение элемента в сетке по вертикали, посредством указания номеров линий. Свойство grid-row объединяет grid-row-start, grid-row-end.
Пример:
grid-row-start: 4;
grid-row-end: 5;
grid-row: 4/5;

grid-area

Свойство grid-area определяет положение элемента в сетке. Свойство grid-area объединяет grid-row, grid-column.
Пример:
grid-area: 1 / 1 / 2 / 2;
grid-area: aside;

Отладка

Выравнивание grid элементов

Свойства для grid контейнера

align-items: stretch | start | end | center;
justify-items: stretch | start | end | center;

Свойства для grid элементов

align-self: stretch | start | end | center;
justify-self: stretch | start | end | center;

Выравнивание grid контейнера

align-content: stretch;
/* start | end | center | space-around | space-between | space-evenly */
justify-content: stretch;
/* start | end | center | space-around | space-between | space-evenly */

Order

Свойство order определяет (переопределяет) порядок следования grid элементов внутри grid контейнера. 
      .grid-item {
        order: 0 | -1 | 1 | 2 ... ;
      }

grid-auto-flow

grid-auto-flow: row;
grid-auto-flow: column;
grid-auto-flow: dense;
grid-auto-flow: row dense;
grid-auto-flow: column dense;
