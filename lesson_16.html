CSS Фреймворк

CSS Фреймворк - это набор готовых решений (элементы UI, сетка, миксины, функции) для ускорения верстки.

Знакомство с фреймворком bootstrap

  • скачать bootstrap;
  • подключить bootstrap;
  • разобраться с работой сетки bootstrap;
  • кастомизировать сетку bootstrap;
  • оптимизировать подключение bootstrap модулей.
Важно: для корректной работы JavaScript модулей bootstrap (< 5v) необходимо подключить библиотеку jquery.

Кастомизация сетки bootstrap

      $grid-columns: 12;
      $grid-gutter-width: 30px;

Кастомизация сетки bootstrap

      $grid-breakpoints: (
        xs: 0,
        sm: 480px,
        md: 768px,
        lg: 1024px
      );

Кастомизация сетки bootstrap

      $container-max-widths: (
        sm: 420px,
        md: 720px,
        lg: 960px
      );
