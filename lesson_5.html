CSS

CSS (Cascading Style Sheets) - это язык, используемый для представления внешнего вида документа (HTML, XML, XHTML, SVG).
CSS отвечает за:
  • стили (внешний вид);
  • визуальные эффекты;
  • анимации.

Каскадность

Каскадность в CSS - это механизм комбинирования стилей
из разных CSS-правил.

Наследование

Наследование в CSS - это свойство, благодаря которому правила форматирования (стили) передаются от родителя к потомку.
Важно: далеко не все свойства в CSS наследуются.

Какие свойства наследуются в CSS?

В основном наследуются свойства, которые касаются стилизации текста: font-size, font-family, font-style, font-weight, color, text-align, text-transform, text-indent, line-height, letter-spacing, word-spacing, white-space, direction

Способы задания стилей элементу

  • тег <style>;
  • атрибут style;
  • тег <link>.

Normalize.css vs Reset.css

normalize.css (reset.css) - это набор стилей для нормализации (сброса) родных стилей тегов

Практика

Задача: нормализовать/сбросить родные стили тегов
используя normalize.css / reset.css.

Селектор

Селектор - это часть CSS-правила, которая сообщает браузеру, к какому элементу (или элементам) веб-страницы будет применён стиль. 
      * { 
        color: red;
      }
* - универсальный селектор

Селектор тег

HTML CSS 
            <h1>
              Заголовок
            </h1>
          
            h1 { 
              color: red;
            }

Селектор id

HTML CSS 
            <h1 id='title'>
              Заголовок
            </h1>
          
            #title { 
              color: red;
            }
Важно: никогда не используйте id для стилизации!

Селектор class

HTML CSS 
            <h1 class='title'>
              Заголовок
            </h1>
          
            .title { 
              color: red;
            }

Стилизация текста

Способы подключения шрифтов

  • Google Fonts
  • @font-face

Практика

Задача: подключить необходимые для макета шрифты используя
сервис Google Fonts.

Подключение через @font-face

      @font-face {
       font-family: 'Roboto';
       font-style: normal;
       font-weight: 400;
       font-display: swap;
       src: local('Roboto Regular'), local('Roboto-Regular'),
            url(Roboto.woff2) format('woff2');
      }

Форматы шрифтов

ttf, eot, svg, woff
woff2

Font Display

Font Display определяет то, как шрифт, подключенный через font face будет отображаться в зависимости от того, загрузился ли он и готов ли к использованию.

Цвета в CSS

Формат Описание Пример
RGB Можно определить цвет, используя значения красной, зеленой и синей составляющей в десятичном исчислении. rgb(0,255,255) rgb(0%,100%,100%)
RGBA Формат RGBA похож по синтаксису на RGB, но включает в себя альфа-канал, задающий прозрачность элемента. rgba(0,255,255, 0.2) rgba(0%,100%,100%, 0.2)
HEX Для задания цветов используются числа в шестнадцатеричном коде. #00ffff
#0ff
HSL Hue (оттенок), Saturate (насыщенность) и Lightness (светлота) hsl(180, 100%, 50%)
HSLA Формат HSLA похож по синтаксису на HSL, но включает в себя альфа-канал, задающий прозрачность элемента. hsla(180, 100%, 50%, 0.2)
