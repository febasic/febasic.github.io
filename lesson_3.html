Тег form

HTML элемент <form> служит тегом-оберткой (контейнером) для всех элементов внутри него. Он объединяет данные со всех элементов в одно целое.
Благодаря элементу <form> мы можем получать данные, вводимые пользователем и отправлять их на сервер.

Атрибуты тега form

Атрибут Назначение
action указывал адрес обработчика формы (пример: action.php)
autocomplete включает/отключает автозаполнение
novalidate включает/отключает HTML валидацию

Элементы форм

тег описание вид
<input> предназначен для ввода данных пользователем
<select> позволяет выбрать пункт из списка
<textarea> поле для ввода нескольких строк текста
дополнительные параметры: cols, rows
<fieldset> объединяет несколько элементов в одну группу
Заголовок группы
<legend> заголовок группы
<button> кнопка отправки/очистки данных
