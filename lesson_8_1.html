Position

Свойсто position определяет как элемент будет позиционироваться в документе относительно других элементов либо относительно самого себя (в случае с position: relative ).

position: static | relative | absolute | fixed | sticky

Position

Конечное положение элемента у которого position отлично от static (значение по умолчанию) определяется с помощью свойств
top right bottom left

Static

static - значение по умолчанию. Для static элементов свойства top right bottom left а также z-index игнорируются.

Relative

Элемент у которого задано свойсто position: relative может быть смещен относительно себя с помощью свойст top right bottom left. При этом такое смещение не будет влиять на положение других элементов.

Absolute

Элемент у которого задано свойсто position: absolute выпадает из нормального потока и позиционируется относительно ближайшего своего родителя у которого position отлично от static.
Положение такого элемента определяется с помощью свойств
top right bottom left.

Fixed

Элемент с postion: fixed ведет себя точно так же как элемент с position: absolute, но позиционируется относительно
элемента body. (есть исключения)

Sticky

Элемент с position: sticky позиционируется в соответствии с нормальным потоком, а затем смещается относительно своего родителя. Смещение не влияет на положение любых других элементов.
