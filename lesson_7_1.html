Float (плавающие элементы)

Свойство float определяет выравнивание элемента по левой или правой стороне, при этом остальные элементы будут обтекать float-элемент (плаващий элемент). Плавающий элемент выпадает из нормального потока. 
      float: left | right | none

Практическое применение

  • обтекание изображения в тексте;
  • модульные сетки.

Clearfix

Clearfix - это вспомогательный класс, служащий для устранения побочных эффектов, вызванных плавающими элементами.

Простейший пример Clearfix

      .clearfix:after {
        content: '';
        display: block;
        clear: both;
      }
