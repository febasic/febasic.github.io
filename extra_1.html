Репозиторий

Первые шаги

  1. установить git на компьютер;
  2. выполнить базовую настройку;
  3. инициализировать репозиторий;
  4. создать первый commit;
  5. зарегистрировать аккаунт на GitHub;
  6. залить код проекта на удаленный репозиторий.

Первоначальная настройка

      git config --global user.name "John Doe"
      git config --global user.email johndoe@example.com
      git config --global core.editor subl
настройки сохраняются в файле ~/.gitconfig

Базовые команды

Команда Описание
git status проверка состояния репозитория
git init инициализация нового репозитория
git add добавить файл(ы) в индекс для отслеживания
git commit сохранить изменения в репозиторий
git push актуализировать удаленный репозиторий
git pull актуализировать локальный репозиторий
git checkout переключение между ветками/коммитами
git clone создание локальной копии удаленного репозитория

.gitignore

Файл .gitignore содержит в себе пути к файлам/папкам, которые следует скрыть от системы контроля версий по тем или иным причинам.
