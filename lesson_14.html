Что такое препроцессор?

CSS препроцессор - это надстройка над CSS, которая расширяет функционал и синтаксис CSS. Код написанный в синтаксисе препроцессора компилируется в CSS.

Препроцессор SASS

SASS - это CSS препроцессор, который существенно расширяет синтаксис и возможности CSS. SASS обладает широким спектром возможностей и функций: переменные, вложенные правила, миксины, функции, циклы, условные опереторы и др.

SASS features

  • удобный и читабельный синтаксис (the indented syntax);
  • возможность разбивать кодовую базу на отдельные фрагменты (@import);
  • переменные, maps (Ассоциативные массивы);
  • циклы (@for, @each);
  • условные операторы (@if, @else)
  • @mixin, @function;
  • вывод ошибок и предупреждений (@debug, @warn, @error).

Вложенные правила

Sass позволяет вкладывать правила CSS друг в друга. Вложенные правила применяются только для элементов, соответствующих внешним селекторам.
SASS CSS 
            .link
              color: black
              .icon
                font-size: 14px
          
            .link {
              color: black;
            }
            .link .icon {
              font-size: 14px;
            }

& в SASS

Амперсанд для родительского селектора позволяет создать новый, расширенный селетор на основе родительского.
SASS CSS 
            .link
              color: black
              &:hover
                color: blue
          
            .link {
              color: black;
            }
            .link:hover {
              font-size: 14px;
            }

Переменные

Переменные в SASS - это способ хранения информации для дальнейшего ее переиспользования в коде проекта. 
      $primary-color: black
      $secondary-color: white
      $primary-font: 'Roboto', sans-serif
      $secondary-font: 'Open Sans', sans-serif

if()

@if

Комбо-пример для if(), @if

      $type: monster;
      p {
        @if $type == ocean {
          color: blue;
        } @else if $type == matador {
          color: red;
        } @else if $type == monster {
          color: green;
        } @else {
          color: black;
        }
      }

Директива @for

      @for $i from 1 through 3 {
       .item-#{$i} { width: 2em * $i; }
      }

Директива @each

      @each $animal in puma, sea-slug, egret, salamander {
        .#{$animal}-icon {
          background-image: url('/images/#{$animal}.png');
        }
      }

Директива @while

      $i: 6;
      @while $i > 0 {
        .item-#{$i} { width: 2em * $i; }
        $i: $i - 2;
      }

Функции
